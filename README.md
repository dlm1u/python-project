# Трекер настроения с элементами дневника

**Описание проекта:**  
Это телеграм-бот, который позволяет отслеживать настроение пользователя и вести личный дневник.  

- **Отслеживание настроения:**  
  Настроение пользователя задаётся с помощью смайликов, а сохраняется как числовая оценка от 1 до 5. Бот собирает статистику и может отображать её за неделю или месяц.

- **Личный дневник:**  
  Пользователь может записывать свои мысли или описывать, как прошёл день. Бот позволяет сохранять эти записи и при необходимости "отдавать" их обратно пользователю.

- **Анализ данных:**  
  Проект включает сбор тестовых данных о настроении пользователей за определённый период. На их основе можно строить графики изменения эмоционального состояния и проводить анализ:
  - среднее настроение по пользователям,
  - изменение настроения по дням,
  - анализ по категориям (пол, возраст).

- **Регистрация пользователей**  
  Пользователи будут указать пол и возраст для сбора более подробной статистики и анализа настроения по категориям.

- **Поднятие настроения:**  
  В бот будет добавлен метод, который показывает мотивирующие картинки или фотографии, чтобы улучшить настроение пользователя.

- **Напоминания:**  
  Настроены ежедневные напоминания, чтобы пользователь не забывал отмечать своё настроение.

---

**Цель проекта:**  
Создать удобный инструмент для ведения дневника настроений и анализа эмоционального состояния пользователей на основе собранных данных.
